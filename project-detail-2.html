TaskMaster AI

Python OpenAI Docker

The Brief: Create a todo list application that prioritizes itself, removing the decision fatigue of "what to do next."

The Logic: Using Natural Language Processing (NLP), the app analyzes the semantic urgency of a task (e.g., "Buy milk" vs "File taxes"). It then re-orders the database entries automatically based on a custom urgency algorithm.

Figure 2.1: The AI agent analyzing task context.

Tech Stack Detail

  • Backend: Flask API with Python.
  • AI: OpenAI GPT-4 API for semantic analysis.
  • Database: PostgreSQL with vector embeddings.
  • Deploy: Docker containers on AWS ECS.
