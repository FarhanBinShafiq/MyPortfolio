NeoFinance

React D3.js WebSockets

The Challenge: Building a professional trading dashboard that could handle 50,000 data points per second without freezing the browser thread or causing jank.

The Solution: I implemented a virtualized list for the order book to only render visible items. For the charts, I bypassed the DOM and used the Canvas API directly via a custom D3 integration. This kept the UI running at a smooth 60fps even under heavy load.

Figure 1.1: The main trading interface showing real-time candlestick charts.

Key Features

  • Real-time WebSocket data stream integration using RxJS.
  • JWT Authentication with 2FA support.
  • Customizable layouts (grid vs list view).
  • Dark/Light mode toggles with persistent state.
